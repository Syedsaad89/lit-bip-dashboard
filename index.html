LIT BIP — Integrated Project Dashboard
Centralised commissioning and progress tracking across all building systems.
Each tab has its own independent live data source, CSV upload, and report download.

🖥️ Crestron Tabs

  • Device Integration & Commissioning — Full KPI dashboard with charts, stage completion, delay tracking, device types, and a filterable detail table. Supports live Apps Script sync.

  • LCP — Lighting Control Panel commissioning progress.

  • Network Racks — Network infrastructure commissioning status.

  • Pool Fibre Lights — Pool fibre lighting system progress.

🏗️ BMS Tab

Building Management System commissioning tracker. Upload or connect a live CSV to view progress and device status for BMS integration work. The generic table renderer auto-detects all CSV columns.

🔗 Connecting Live Data

  1. Open your Google Sheet
  2. Go to Extensions → Apps Script
  3. Paste the doGet script
  4. Deploy as Web App → Access: Anyone
  5. Copy the /exec URL
  6. Click 🔗 Source on any tab and paste it

📄 Reports & Downloads

  • ↑ Update CSV — Upload a new CSV to refresh the active tab

  • 🔗 Source — Connect a live URL with optional auto-refresh (1–30 min)

  • 📄 Report — Printable PDF summary of the active tab

  • ⬇ Download — Full dashboard as standalone HTML file
